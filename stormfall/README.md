# Stormfall

Stormfall is a playable top-down co-op survival prototype set in Scrapyard Forest after a massive tornado.

## Recommended Stack

For a beginner-friendly prototype, the best practical stack is plain HTML, CSS, and JavaScript Canvas:

- No engine install is required.
- It runs locally in any modern browser.
- The code is small enough to learn from and modify.
- The core game loop can later move to Phaser, Godot, or Unity when online multiplayer, art pipelines, and larger levels become important.

## How To Run

Open `index.html` in a browser.

## Controls

- Player 1: `WASD` move, `E` gather/build/revive, `F` attack, `C` craft
- Player 2: arrow keys move, `/` gather/build/revive, `.` attack, `,` craft

## Prototype Features

- Main menu
- One playable level: Scrapyard Forest
- Second playable region: Burning Plains
- Larger scrolling wilderness map with a camera
- Two-player local co-op foundation
- Player movement and health
- Downed/revive system
- Camp respawn with gear loss
- Shared team resources and player inventories
- Resource gathering
- Camp building
- Basic crafting
- Easier tornado spirit enemy AI with shorter chase range and lower damage
- Robot Cat boss fight
- Flying Fire Shark boss fight
- Simple quest chain and level completion state
- More human-looking placeholder survivor characters
- Improved Canvas graphics: richer terrain, burned trees, fire patches, better pickups, and boss silhouettes

## Future Levels

- Final Level: Flooded Ruins, boss Water Dragon

## Camp Building Roadmap

The prototype includes shelter, crafting bench, medical station, and walls. Future camp structures can include watch towers, weapon forges, food farms, rain collectors, storage crates, garages, generators, campfires, and training areas.

No drone launch pads are planned.
